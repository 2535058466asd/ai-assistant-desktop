# Nova - UI 优化清单

> 品牌名：**Nova**（新星）
> 基于现有代码分析 + 设计稿对比，按优先级排列
> 现有 UI 已有毛玻璃/渐变基础，不需要重做，在现有基础上精修即可

## P0 - 核心体验（优先做）

### 0. 设置页（抽屉式）— 所有配置统一入口
- **现状**: 模型管理、语音设置等散落在 Header 和各处，没有统一设置页
- **设计决策**: Header 只保留"模型快速切换"和"亮/暗色切换"，其余全部移入设置页
- **交互方式**: 右侧抽屉面板（从侧边栏底部设置图标触发），用户可边聊天边调设置
- **设置页板块**:
  ```
  ⚙️ 设置
  ├── 🤖 模型管理        — 模型列表增删排序、默认模型设置
  ├── 🔑 API 配置        — 各平台 API Key 管理
  ├── 🧠 记忆管理        — 记忆列表查看/搜索/删除/导入导出
  ├── 🎤 语音设置        — ASR/TTS 引擎选择、语速、音色
  ├── 🔍 搜索设置        — 搜索引擎配置
  ├── ⌨️ 快捷键          — 自定义快捷键
  └── ℹ️ 关于            — 版本信息、更新日志
  ```
- **Header 精简后只保留**:
  - 左：侧边栏切换 + Logo + 品牌名
  - 中：模型快速切换下拉
  - 右：亮/暗色切换 + 语音对话按钮
- **涉及文件**: 新建 `Settings/SettingsDrawer.tsx` + `SettingsDrawer.module.css`，修改 `Header.tsx`（移除多余按钮）

### 1. Markdown 渲染优化
- **现状**: 已引入 react-markdown + remark-gfm，但 AI 回复长文本时排版差
- **优化**:
  - 表格样式美化（交替行背景、圆角边框）
  - 代码块样式（语法高亮、复制按钮、语言标签）
  - 引用块样式（左侧渐变边框 + 浅色背景）
  - 列表缩进和间距调整
- **涉及文件**: `ChatArea.tsx`（渲染逻辑）、`ChatArea.module.css`（新增 markdown 子样式）

### 2. 消息流式输出体验
- **现状**: 有流式光标和蓝色右边框，但缺少平滑感
- **优化**:
  - 代码块流式输出时先显示骨架/占位，渲染完再展示
  - 长文本流式输出时自动滚动到底部（平滑滚动，不是跳动）
  - 流式输出结束后移除右边框，加一个淡出过渡
- **涉及文件**: `ChatArea.tsx`、`ChatArea.module.css`

### 3. 输入框体验提升
- **现状**: 基本功能完整，但细节不够
- **优化**:
  - 自适应高度（textarea 根据内容自动增长，已有 max-height: 120px）
  - `Ctrl+Enter` / `Cmd+Enter` 发送（目前是 Enter 发送）
  - 输入框为空时发送按钮更明显的禁用态
  - 粘贴图片时显示预览缩略图（为后续图片输入做准备）
- **涉及文件**: `InputArea.tsx`、`InputArea.module.css`

## P1 - 视觉精致度（其次做）

### 4. 背景动态光晕
- **现状**: 有两个固定位置的 radial-gradient 光晕
- **优化**:
  - 光晕随时间缓慢移动/呼吸（CSS animation 或 JS）
  - 亮色模式下光晕更柔和
  - 可选：根据当前对话主题/情绪微调光晕颜色
- **涉及文件**: `AppLayout.module.css`、`global.css`

### 5. 侧边栏交互增强
- **现状**: 对话项有悬停效果和选中态，但功能单一
- **优化**:
  - 对话项右键菜单（重命名、置顶、删除）
  - 对话列表拖拽排序（可选，优先级低）
  - 搜索框输入时实时过滤对话列表
  - 新建对话按钮点击时的微动画（图标旋转 + 边框变实线）
- **涉及文件**: `Sidebar.tsx`、`Sidebar.module.css`

### 6. 主题切换过渡动画
- **现状**: 通过 `data-theme` 属性切换，颜色直接变化
- **优化**:
  - 切换时所有颜色属性加 `transition: background-color 0.3s, color 0.3s, border-color 0.3s`
  - 切换按钮加旋转动画（太阳/月亮 图标旋转切换）
  - 光晕颜色同步过渡
- **涉及文件**: `global.css`（全局 transition）、`Header.tsx`（切换按钮动画）

### 7. AI 消息气泡精致化
- **现状**: 毛玻璃 + 悬停渐变边框，已经不错
- **优化**:
  - 悬停时边框渐变更明显（从几乎不可见到可见）
  - 操作按钮（复制/重试）加 tooltip 提示文字
  - AI 头像加微妙的呼吸动画（在线状态感）
- **涉及文件**: `ChatArea.module.css`

## P2 - 交互亮点（锦上添花）

### 8. 欢迎页增强
- **现状**: Logo + 标题 + 4 张快捷卡片，有毛玻璃效果
- **优化**:
  - Logo 加微妙的 3D 旋转跟随鼠标效果
  - 快捷卡片悬停时图标微微放大
  - 底部加一行小字 "按 / 快速搜索命令"
- **涉及文件**: `WelcomeScreen.tsx`、`WelcomeScreen.module.css`

### 9. 命令面板（Cmd+K / Ctrl+K）
- **新增功能**:
  - 弹出浮层，快速搜索对话、切换模型、清空当前对话、导出记录
  - 类似 Cursor 的命令面板风格
  - 毛玻璃背景 + 模糊搜索
- **涉及文件**: 新建 `CommandPalette.tsx` + `CommandPalette.module.css`

### 10. 消息操作增强
- **现状**: AI 消息有复制按钮
- **优化**:
  - 用户消息也加复制按钮（悬停显示）
  - AI 消息加"重新生成"按钮
  - 长消息加"收起/展开"功能
  - 复制成功后按钮变勾 + 1.5秒后恢复
- **涉及文件**: `ChatArea.tsx`、`ChatArea.module.css`

### 11. 滚动条美化
- **现状**: 已有自定义滚动条样式
- **优化**:
  - 聊天区滚动条在非悬停时完全隐藏，悬停时淡入
  - 滚动条宽度从 6px 减到 4px（更精致）
- **涉及文件**: `global.css`

### 12. 加载与空状态
- **优化**:
  - 首次加载时加一个简洁的 splash screen（Logo + 加载进度）
  - 网络断开时输入框上方显示"网络连接已断开"提示条
  - 模型切换时加短暂的 loading 状态指示
- **涉及文件**: `App.tsx`、`InputArea.tsx`
