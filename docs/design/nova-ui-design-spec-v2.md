| 入口 | 功能 |

| 设置抽屉 | 模型管理、API 配置、记忆管理、语音设置、搜索设置、快捷键 |
| Header 右侧按钮 | 模型切换、主题切换 ☀️/🌙 |
| 输入框区域 | 📎 附件上传、🎤 语音转文字、🎙️ 语音对话、➤ 发送 |

## 六、语音动画统一（P1）

### 问题
语音转文字和语音对话模式的动画风格不统一。

### 统一方案
共享同一套颜色系统和动画风格：

| 状态 | 颜色 | 动画 |
|------|------|------|
| 录音中 | 红色 (#ef4444) | 脉冲波纹扩散 |
| 监听中 | 绿色 (#22c55e) | 呼吸闪烁 |
| 思考中 | 蓝色 (#3b82f6) | 脉冲缩放 |
| 播放中 | 橙色 (#f97316) | 弹跳动画 |

### 语音转文字（轻量）
- 麦克风图标变红色
- 图标周围加脉冲波纹
- 输入框边框变红色

### 语音对话模式（完整）
- 底部状态栏显示当前状态
- 状态指示点用对应颜色 + 动画
- 整体风格与语音转文字一致

### 涉及文件
- `InputArea.tsx` + `InputArea.module.css` — 语音转文字 + 语音对话动画统一

## 七、搜索对话优化（P2）— ⚠️ 可能有问题

### Bug 4：搜索对话功能可能异常
**现状**：搜索框存在但不确定是否能正常过滤对话列表
**修复**：
- 确认输入时**实时过滤**对话列表
- 搜索图标点击时自动聚焦搜索框
- 无匹配结果时显示"未找到相关对话"

### 方案
- 搜索框已有，确认输入时**实时过滤**对话列表
- 搜索图标点击时自动聚焦搜索框
- 无匹配结果时显示"未找到相关对话"

### 涉及文件
- `Sidebar.tsx` — 搜索过滤逻辑确认/优化
