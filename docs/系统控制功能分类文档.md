## 🎯 三、统一接口设计

### 所有操作都通过一个技能调用

```
universal-control 技能
│
├── action: "open"           → 应用管理
├── action: "open-folder"    → 文件管理
├── action: "volume-up"      → 媒体控制
├── action: "mute"           → 媒体控制
├── action: "lock-screen"     → 系统控制
├── action: "shutdown"        → 系统控制
└── ...                      → 更多
```

### 参数结构

```typescript
{
  // 必填：操作类型
  action: string,
  
  // 可选：目标对象
  target?: string,          // 应用名、文件夹名等
  
  // 可选：详细参数
  params?: {
    query?: string,         // 搜索关键词（如歌曲名）
    volume?: number,        // 音量值
    path?: string,          // 文件路径
    delay?: number,         // 延迟时间（秒）
  }
}
```

## 🔧 五、Electron API 对应关系

### 渲染进程调用 → 主进程执行

```typescript
// universal-control/index.ts（渲染进程）
case 'open':
  result = await api.systemOpenApp(target);
  break;

case 'open-folder':
  result = await api.systemOpenFolder(target);
  break;

case 'volume-up':
  result = await api.systemAdjustVolume(params.volume || 10);
  break;

case 'mute':
  result = await api.systemMute();
  break;

case 'lock-screen':
  result = await api.systemLockScreen();
  break;

case 'shutdown':
  result = await api.systemShutdown();
  break;
```

### 主进程 IPC 处理器

```typescript
// src/main/index.ts（主进程）
ipcMain.handle('system:open-app', (event, appName) => {
  return systemControl.openApp(appName);
});

ipcMain.handle('system:open-folder', (event, folderName) => {
  return systemControl.openFolder(folderName);
});

ipcMain.handle('system:adjust-volume', (event, delta) => {
  return systemControl.adjustVolume(delta);
});

ipcMain.handle('system:mute', () => {
  return systemControl.toggleMute();
});

ipcMain.handle('system:lock-screen', () => {
  return systemControl.lockScreen();
});

ipcMain.handle('system:shutdown', () => {
  return systemControl.shutdownComputer();
});
```
